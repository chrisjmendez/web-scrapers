# ScrapingBee


## Getting Started

### Step 0 - Register with ScrapingBee

[Register with ScrapingBee](https://www.scrapingbee.com?fpr=chris-m37)


### Step 1 - Configure ```.env```

Configure ```.env``` and 

```
nano .env
```

Within ```.env```
```
SCRAPING_BEE_API=PUT YOUR API KEY HERE
```

### Step 2 - Download libraries

```
npm i
```
